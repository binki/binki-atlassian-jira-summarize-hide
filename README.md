Jira will display an annoying in-page “AI-powered” comment summary and button to open this summary. This hides both of those.

Some Jira sites are configured to pre-expand the comment summary. This pre-expanded summary will change size dynamically after the page is loaded, preventing use of the page during this time. This fixes that issue.

This is just some CSS but I don’ t really have a system for deploying fixes like this set up for anything other than Greasemonkey.
